# Bộ Style Preset

12 preset phong cách thị giác cho PPTX. Mỗi preset định nghĩa màu nền, màu chữ, màu nhấn và cặp font. Chọn dựa trên đối tượng và nội dung.

---

## 1. cleanCorporate
**Cảm giác:** Sáng, chuyên nghiệp, đáng tin
**Phù hợp với:** Báo cáo doanh nghiệp, trình bày B2B, hội đồng quản trị

- Nền: `#ffffff` / `#f8f9fa`
- Chữ: `#212529` / `#6c757d`
- Nhấn: `#4361ee` (xanh lam) / `#3a0ca3` (xanh đậm)
- Font: Plus Jakarta Sans (tiêu đề) + Source Sans 3 (nội dung)

## Tạo theme tùy chỉnh

Khi user yêu cầu phong cách không có trong 12 preset (ví dụ: "cute pink", "xanh mint", "vintage vàng"...), dùng `createTheme()` với `ThemeConfig` tùy chỉnh:

```typescript
// Tất cả màu là hex string KHÔNG có dấu #
const t = createTheme({
  bg: { primary: '...', secondary: '...' },
  text: { primary: '...', secondary: '...' },
  accent: '...',
  accentSecondary: '...',
  font: { heading: '...', body: '...', mono: 'Consolas' },
});
```

Bỏ qua bất kỳ trường nào để giữ giá trị mặc định.

### Ví dụ: Cute Pink

Khi user yêu cầu phong cách hồng, nữ tính, pastel:

```typescript
const t = createTheme({
  bg: { primary: 'fff0f5', secondary: 'ffe4ef' },
  text: { primary: '3d1a2e', secondary: '8b4a6b' },
  accent: 'e91e8c',
  accentSecondary: 'ff9ec4',
  font: { heading: 'Fraunces', body: 'Outfit', mono: 'Consolas' },
});
```
