## Principle: 1 Slide = 1 Idea
- Each slide conveys one message only.
- Slide title = that message (not a topic label).
  - Wrong: "Q4 Revenue"
  - Correct: "Q4 Revenue Hit Highest Point of the Year"

## Bullet Points
- Maximum 5 bullets per slide.
- Each bullet <= 10 words.
- Start with a number or action word when possible.
  - Correct: "Q4: +340% vs Q3"
  - Correct: "Investigate 3 negative records from August"

## Standard Deck Structure
- Slide 1: Title + Presenter name + Date
- Slide 2: Agenda / Topics to be covered
- Slides 3 → N-1: Main content
- Slide N: Conclusion + Next actions
- Final slide: "Q&A"

## Converting from Report to Slides
- Executive Summary → Summary results slide
- Each Key Finding → 1 slide
- Recommendations → 1 "Action Items" slide
- Skip "Detailed Analysis" sections from the report — too long for slides
