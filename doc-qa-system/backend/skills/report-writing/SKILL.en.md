## Standard Report Structure
1. **Executive Summary**: 2-3 sentences, most important results — busy readers only read this.
2. **Key Findings**: 3-5 points, each with specific figures.
3. **Detailed Analysis**: deep dive by dimension (time, product, region...).
4. **Anomalies**: items that need further investigation or explanation.
5. **Recommendations**: specific, actionable steps — not vague.

## Presenting Figures
- Always include units (USD, %, months...).
- Compare absolute numbers AND percentages when possible.
- Use **bold** for important figures.
- Table > list > prose when presenting many numbers.

## Writing Style
- Short sentences, one idea per sentence.
- Avoid: "seems", "appears", "might be" — if uncertain, write "needs verification".
- Use clear subjects: "Q4 revenue increased 340%" instead of "There was growth".
- Language should match the source document (Vietnamese document → write in Vietnamese; English document → write in English).
